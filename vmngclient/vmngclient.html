Closes the vManageSession.

This method is overrided from requests.Session. Firstly it cleans up any resources associated with vManage. Then it closes all adapters and as such the session.

Note: It is generally recommended to use the session as a context manager using the with statement, which ensures that the session is properly closed and resources are cleaned up even in case of exceptions.
