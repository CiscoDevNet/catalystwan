class vmngclient.session.Session(url: str, username: str, password: str, port: Optional[int], subdomain: Optional[str] = None, timeout: int = 30)¶

Bases: object

Base class for API sessions for vmanage client.

Defines methods and handles session connectivity available for provider, provider as tenant, and tenant.

Parameters:

  • url – IP address or domain name, i.e. ‘10.0.1.200’ or ‘fruits.com’

  • port – port

  • username – username

  • password – password

  • timeout – timeout

wait_for_server_reachability(retries: int, delay: int, initial_delay: int = 0) → bool¶

Checks if vManage API is reachable by sending server request.

Retries on HTTPError for specified number of times. Delays between each request are configurable, It is intended to be used as a probe, so it doesn’t raise original error from exception.

Parameters:

  • retries – total number of retires

  • delay – time to wait between each retry

  • initial_delay – time before sending first request

Returns:

True if device is reachable, False if not

Return type:

Bool
