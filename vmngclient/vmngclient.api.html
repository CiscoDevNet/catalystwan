check_alarms(expected: List[Dict[str, str]], timeout_seconds: int = 240, sleep_seconds: int = 5) → Dict[str, set]¶

Checks if alarms have occurred.

For a specified period of time, the method queries the API about unread alarms and checks if they occur in the expected ones. If all alarms occur before the time expires, it will not poll any further.

Parameters:

  • expected (list) – The list of expected alarms.

  • timeout_seconds (int) – Failure timeout.

  • sleep_seconds (int) – Sleep time.

Returns:

The dictionary with alarms that occurred (key ‘found’) and did not (key ‘no-found’)
